Image Processing 101

Due to the heterogeneous coding experience in our team we decided to dedicate a considerable amount of our time to pair coding and the basics of image processing. As for the choice of our coding language, we decided that both Matlab and Python are sound choices with their respective advantages and disadvantages.

Topics covered were amongst others:

  1. Basic file I/O and manipulation
  2. Editing the exposure of images
  3. Examplary filters
  4. Point, local and global operators
  5. A basic machine learning example using SVM
